What this club does:

- Builds useful open source projects for school and community
- Runs 4-week seasons with two teams
- Practices coding, design, planning, documentation, testing, and presentation

Core rules:

- Public code only
- No student personal data or sensitive school data in public repos
- Go, JavaScript, and TypeScript are the allowed implementation languages
- MIT is the default license
